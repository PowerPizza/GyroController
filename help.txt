using configs.json you can change the following settings of driver
1. sensitivity-X = it must be a positive integer [Default = 200]
2. sensitivity-y = it must be a positive integer [Default = 160]
3. sensorCoolDown = it must be in seconds [Default = 0.2]; 0.2 seconds = 200 milliseconds

this driver is for a spacial hardware name gyroController and hardware must require.
